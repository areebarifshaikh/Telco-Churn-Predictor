# Telco Customer Churn Predictor

A machine learning classification model that predicts customer churn for telecom companies.

## Features
- Binary classification using scikit-learn
- Trained on customer demographics and service usage
- Interactive Streamlit web interface for predictions
- Model preprocessing and feature scaling included

## Installation
```bash
pip install -r requirements.txt
```

## Run Locally
```bash
streamlit run app.py
```

## Deployment
Deployed on [Streamlit Community Cloud](https://telco-churn-predictor-12.streamlit.app)
